O gerenciador de pastas é um programa para gerenciar em lote operações com pastas(diretórios) do sistema windows.

Com ele podemos:
<br>
Excluir pastas que estão vazias<br>
Renomear pastas <br>
Adicionar, criar pastas na quantidade que queremos<br>
Alterar de caixa alta para caixa baixa os nomes das pastas<br>
Tudo em lote, economizando tempo.<br>
<br>

Para usar basta instalar as dependecias do arquivo requirements.txt
e rodar o main.py.
<br>
Certifique que todos os arquivos estejam na mesmo diretório.
